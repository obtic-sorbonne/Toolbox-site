Tanagra

Tanagra est un outil de cartographie des lieux dans les textes. Il existe deux versions de l'outil : la première est une interface qui permet uniquement d'identifier et de cartographier les lieux mentionnés dans un texte. La seconde version est enrichie de l'analyse de sentiments et permet d'identifier la polarité associée aux lieux mentionnés.

  • Tanagra v1 : Cartographie des lieux dans les textes [ interface ]
  • Tanagra v2 : Analyse de sentiments et cartographie des lieux dans les textes [ notebook ]
