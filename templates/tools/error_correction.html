Deux types de sortie sont possibles pour l'outil de correction d'erreurs :

  • Case non cochée : les corrections effectuées par l'outil apparaitront ainsi **correction**, afin de pouvoir observer comment les éléments ont été changés. Il vous sera donc possible de vérifier la correction, manuellement ou en utilisant, par exemple, notre outil "Comparaison" et de changer si nécessaire. Par la suite, vous pourrez enlever ces marques dans l'outil "Nettoyage de texte".
  • Case cochée : les corrections effectuées ne seront pas visibles dans la version de sortie, ce qui permet une utilisation directe du fichier ensuite, avec cependant une correction non vérifiée
