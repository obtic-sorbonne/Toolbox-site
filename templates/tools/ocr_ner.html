{{ _("Numérisation et reconnaissance d'entités nommées") }}

{{ _("L'outil Tesseract permet de numériser des fichiers imprimés") }}. {{ _("L'outil spaCy permet de faire la reconnaissance d'entités nommées") }}.

Cet outil ne vise qu'à fournir les entités nommées d'un document directement transcrit sans correction intermédiaire. Le contenu même du texte ne sera pas fourni dans le fichier de sortie, mais seulement les entités nommées. Si plusieurs images ou PDFs sont envoyés, cela sera interprêté comme un tout.

La sortie spaCy sera au format TXT et apparaitra ainsi : T9 LOC 179 188 Colombine

  • T9 est l’identifiant interne de l’entité (la 9ᵉ entité trouvée dans le document).
  • LOC — le label attribué par spaCy
  • 179 188 — les offsets de caractères dans le texte :
    • 179 = position de début
    • 188 = position de fin
  • Colombine — la chaîne de texte reconnue comme entité.

Les classes d'étiquettes du NER sont les suivantes :

  • PER/PERSON — personne
  • LOC — lieu
  • ORG/GPE — organisation/entités géopolitiques (pays, villes)
  • MISC — divers (événements, œuvres, gentilés, etc.)
  • NORP — nationalités, groupes religieux/politiques
  • etc.
